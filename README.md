# Akıllı Kredi Sistemi 🏦

Bu proje, kullanıcıların finansal verilerini analiz ederek kredi uygunluk durumlarını değerlendiren ve kişiselleştirilmiş kredi teklifleri sunan bir **Java** tabanlı konsol uygulamasıdır. 

Gazi Üniversitesi Bilgisayar Mühendisliği eğitimim sürecinde geliştirdiğim bu çalışma, temel programlama prensiplerini, dosya işlemlerini ve nesne yönelimli programlama (OOP) mantığını içermektedir.

## 🚀 Özellikler

- **Kullanıcı Kayıt ve Giriş:** Kullanıcı bilgilerinin güvenli bir şekilde alınması.
- **Finansal Analiz:** Gelir düzeyi, mevcut borç durumu ve kredi skoru gibi parametrelerin değerlendirilmesi.
- **Kredi Hesaplama:** Algoritmik olarak geri ödeme planlarının ve faiz oranlarının belirlenmesi.
- **Veri Yönetimi:** Kullanıcı verilerinin ve kredi geçmişinin `.txt` dosyaları üzerinden kalıcı olarak saklanması.
- **Hata Yönetimi:** Geçersiz girişlere karşı kullanıcıyı yönlendiren kontrol mekanizmaları.

## 🛠 Kullanılan Teknolojiler

- **Dil:** Java
- **Veri Saklama:** File I/O (Txt tabanlı veritabanı simülasyonu)
- **IDE:** IntelliJ IDEA

## 📋 Kurulum ve Çalıştırma

1. Projeyi bilgisayarınıza klonlayın:
   ```bash
   git clone [https://github.com/egepehlivan4/Akilli-Kredi-Sistemi.git](https://github.com/egepehlivan4/Akilli-Kredi-Sistemi.git)
```

2.  Proje dizinine gidin:
    ```bash
    cd Akilli-Kredi-Sistemi
    ```
3.  Java yüklü olduğundan emin olun ve `src` klasöründeki ana sınıfı (Main) derleyerek çalıştırın.

## 📂 Dosya Yapısı

  - `src/`: Projenin kaynak kodları.
  - `Kisiler.txt`: Sisteme kayıtlı kullanıcıların bilgilerini içeren veri dosyası.
  - `.idea/`: IDE yapılandırma dosyaları.
